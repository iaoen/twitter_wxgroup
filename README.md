## 简介

### twitter/推特 爬取微信群聊二维码

### 群聊码识别使用 opencv 对接微信开源模型

### 当然，QQ 群码也是可以识别的哈，这需要你自己对项目进行简单的修改。

## 注意

#### opencv 版本 必须 4.5.2.54

## 填写配置文件

#### 打开 config.py，参数为 x 的需要自己填写。

## 安装依赖

`pip install pyzbar opencv-python==4.5.2.54 opencv-contrib-python==4.5.2.54`

### 运行程序

`python twimgs.py`
